# Power BI E-Commerce Sales Dashboard

This project is an interactive **Power BI dashboard** built to analyze e-commerce business performance across different countries, states, markets, categories, and shipping modes.

## Project Overview

The dashboard provides a clear overview of key business metrics such as:

- Total Sales: 2.84M
- Total Profit: 405.46K
- Total Quantity Sold: 21K
- Total Shipping Cost: 230.77K

It also helps users explore:

- Sales by country
- Sales by state
- Sales by category
- Sales by market
- Sales by ship mode

## Dashboard Highlights

### Key KPIs
The dashboard presents high-level KPIs for sales, profit, quantity, and shipping cost to give a quick understanding of business performance.

### Geographic Analysis
Users can analyze sales distribution by country and state to identify top-performing regions.

### Category Insights
The dashboard allows filtering by product category such as Furniture, Office Supplies, and Technology.

### Market Analysis
Users can compare performance across different markets including APAC, EU, US, LATAM, Africa, Canada, and EMEA.

### Shipping Analysis
Sales are also analyzed by shipping mode, including Standard Class, Second Class, First Class, and Same Day.
